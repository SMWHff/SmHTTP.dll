命令名称： GetCookieByName 获取指定Cookie的值
命令功能： 获取指定Cookie的值
命令参数：
参数1【必选】：文本型，所有Cookies
参数2【必选】：文本型，Cookie名
返回值： 文本型，返回Cookie值
脚本例子： 
Import "SmHTTP.dll"
Set SmHTTP = CreateObject("SMWH.SmHTTP")

' 开启自动识别对应传入参数
Call SmHTTP.SetAutoParamArray(True)

Dim HTML, Cookies

' 百度搜索
HTML = SmHTTP.HTTP_GET("https://ug.baidu.com/mcp/pc/pcsearch")

' 获取所有Cookies
Cookies = SmHTTP.GetCookies()

' 获取指定Cookie值
TracePrint SmHTTP.GetCookieByName(Cookies, "BAIDUID")
